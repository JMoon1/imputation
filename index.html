    About the package

    Imputation is the process of filling missing values in a dataset with a intelligently reconstructed value. This R package provides simple methods for imputing a data matrix. To install, open a R session and issue

    install.packages("imputation")

    Usage

    
require(imputation)
x = matrix(rnorm(100),10,10)
x[1,1] = NA
x[5,8] = NA
x[2,3] = NA
x[7,7] = NA
k = cv.SVDImpute(x)$k
SVDImpute(x, k)

    Algorithms

    Mean Imputation

    Simply fills missing values using the mean of the column. The mean is the minimizer of squared error loss for i.i.d data

    Persistence Imputation

    For time series data, fill an observation at time \(t\) using the most recent known value

    kNN Imputation

    Find neighbors that don't have missing values, and fill using the mean of the neighbors.

    SVD Imputation

    Least squares can be computed even when there is missing data. The SVD problem can be formulated as \( \DeclareMathOperator*{\argmin}{arg\,min} ||X - UDV^T||_F^2 \) minimizing the squared error between a matrix \(X\) and its low rank projection \(\hat{X}\). This problem can be solved using alternating least squares, and \( \hat{X} = UDV^T \) is returned as the imputed matrix.

    SVT Imputation

    Singular Value Thresholding a la Emmanuel Candes.

    GBM Imputation

    Uses gradient boosting machines, an ensemble of decision trees, to fill missing values. GBM is particularly powerful because it can handle categorical data as well
